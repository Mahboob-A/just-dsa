# Arrays


## Static Array

Insertion on end  - `O(1)`

Insertion on middle - `O(n)`

Deletion on end - `O(1)`

Deletion on middle - `O(n)`

Reading - `O(1)`


## Dynamic Array

Insertion on end  - `O(1)`

Insertion on middle - `O(n)`, amotized time complexity is `O(1)`

Deletion on end - `O(1)`

Deletion on middle - `O(n)`

Reading - `O(1)`
